# 🤖 AI CSS Generator - Llama 3.3 & Groq

Este é um projeto de ferramenta interativa que utiliza Inteligência Artificial para transformar descrições em linguagem natural em código CSS funcional e visualizável em tempo real. Desenvolvido para demonstrar habilidades em integração de APIs de IA e manipulação dinâmica de DOM.

## 🚀 Demonstração
Você pode descrever efeitos como:
- "Um quadrado neon que pulsa em azul"
- "Uma bola saltitante com sombra suave"
- "Um botão com efeito de vidro (glassmorphism)"

## 🛠️ Tecnologias Utilizadas
- **HTML5 & CSS3 Moderno:** Layout estruturado com Flexbox e animações customizadas.
- **JavaScript (ES6+):** Lógica de consumo de API assíncrona (Async/Await) e injeção dinâmica de estilos.
- **Groq Cloud API:** Utilização do modelo **Llama 3.3 70b** para geração de código de alta performance e baixa latência.

## 💡 Funcionalidades Principais
- **Preview Instantâneo:** O código gerado pela IA é aplicado automaticamente a um elemento na tela para visualização imediata.
- **Engenharia de Prompt:** Sistema configurado para garantir que a IA retorne apenas CSS limpo e compatível com as classes do projeto.
- **Cópia em um Clique:** Botão dedicado para copiar o código gerado diretamente para a área de transferência.
- **Feedback de UX:** Spinner de carregamento para indicar o processamento da IA, melhorando a experiência do usuário.
